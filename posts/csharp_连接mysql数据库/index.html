C#_连接MySQL数据库

连接MySQL数据库

02-08-2022

  • CSharp
  • MySQ

先添加MySQL的DLL引用

在MySQL的安转目录下有 MySql.Data.dll 一般有多个版本 自己选择

在vs 项目->添加项目引用->浏览 添加MySql.Data.dll的引用

再在代码中调用

引用

    using MySql.Data.MySqlClient;

建立连接

    // 服务器地址；端口号；数据库；用户名；密码
    public static string connectStr = "server=localhost;port=3306;database=****;user=root;password=****";
    MySqlConnection con = new MySqlConnection(connectStr);

连接

    if (this.con.State == System.Data.ConnectionState.Closed)
    {
        try
        {
            //连接数据库
            this.con.Open();
            Console.WriteLine("连接成功");
        }
        catch (Exception e)
        {
            Console.WriteLine(e.ToString());
        }
        //finally
        //{
        //    // 关闭连接
        //    con.Close();
        //    Console.WriteLine("数据库已关闭");
        //}
    }

SQL语句查询

使用 MySqlCommand（数据不会存储，需要定义一个数组手动存储）
使用适配器 MySqlDataAdapter（将数据存储到内存中，占用内存）
使用事务 BeginTransaction（此方法用来批量执行一组命令，一个命令错误就会全部回滚，防止误提交问题）
MySqlDataAdapter
    try
    {
        DataSet ds = new DataSet();
        MySqlDataAdapter daa = new MySqlDataAdapter("SQL语句", con);
        daa.Fill(ds);
        return ds;
    }
    catch
    {
        return null;
    }
