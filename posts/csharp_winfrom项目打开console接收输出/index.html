C#_WinFrom项目打开Console接收输出

WinFrom项目打开Console接收输出

02-08-2022

  • WinForm
  • CSharp

在Program.cs Class开头引入


    using System.Runtime.InteropServices;

    [DllImport("kernel32.dll")]
    public static extern Boolean AllocConsole();
    
    [DllImport("kernel32.dll")]
    public static extern Boolean FreeConsole();

调用AllocConsole()开启

调用FreeConsole()关闭

代码

using /....../

using System.Runtime.InteropServices;

namespace ......
{
    static class Program
    {
        /// <summary>
        ///  The main entry point for the application.
        /// </summary>
        /// 
        [DllImport("kernel32.dll")]
        public static extern Boolean AllocConsole();
        [DllImport("kernel32.dll")]
        public static extern Boolean FreeConsole();

        [STAThread]
        static void Main()
        {
            AllocConsole();

            /....../

            FreeConsole();
        }
    }
}
